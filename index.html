以下内容为选摘：
Core Data 是iOS SDK 里的一个很强大的框架，允许程序员以面向对象的方式储存和管理数据。使用Core Data框架，程序员可以很轻松有效地通过面向对象的接口管理数据

Core Data是一个模型层的技术。帮助建立代表程序状态的模型层，Core Data也是一种持久化技术，能将模型对象的状态持久化到磁盘，但它最重要的特点是：Core Data 不仅是一个加载、保存数据的框架，它还能和内存中的数据很好的共事

在数据操作过程中，无需编写任何 SQL 语句 Core Data 使用包括实体和实体间关系，以及查找符合某些条件实体的请求等内容

开发者可以在纯对象层上查找与管理这些数据，而不必担心存储和查找的实现细节
Core Data 框架最早出现在 Mac OS X 10.4 Tiger 与 iOS 3.0 系统，经过成千上万的应用程序以及数以百万用户的反复的验证，Core Data 确实已经是一套非常成熟的框架

CoreData 利用了 Objective-C 语言和运行时，巧妙地集成了 Core Foundation 框架。是一个易于使用的框架，不仅可以优雅地管理对象图，而且在内存管理方面表现异常优异

发表于 2015-12-24   |   分类于 iOS凿壁偷光

关于闭包的定义：

闭包是功能性包自包含模块，可以再代码中传递和使用。类似C和OC中的block，闭包可以捕获和存储其所在上下文中任意常量和变量的引用。

闭包有三种形式：

  1. 全局函数是一个有名字但不会捕获任何值的闭包。
  2. 嵌套函数是一个有名字并可以捕获到其封闭函数域内的值的闭包。
  3. 闭包表达式是一个利用轻量级语法所写的，可以捕获其上下文中变量或常量值的匿名闭包。

主要用于异步操作完成后代码的回调，网络访问结果一参数形式传递给调用方。在使用中，闭包会预先定义好一个代码块。

iOS基本动画

发表于 2015-11-28   |   分类于 iOS凿壁偷光

基本动画 CABasicAnimation

KVC

keyPath实际上使用的是KVC的设计模式，keyPath本身的值是属性.后面的属性。

CABasicAnimation 属性

  • fromValue

fromValue is non-nil. Interpolates between fromValue and the current presentation value of the property.

指的是动画会从当前位置跳转到fromValue设值的位置再跳回来。

  • toValue

toValue is non-nil. Interpolates between the current value of keyPath in the target layer’s presentation layer and toValue.

指的是动画会直接跳转到指定的位置。

  • byValue

byValue is non-nil. Interpolates between the current value of keyPath in the target layer’s presentation layer and that value plus byValue.

指的是动画会从当前位置跳到当前位置叠加值后的位置。
