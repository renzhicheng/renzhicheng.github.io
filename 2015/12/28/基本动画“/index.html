###基本动画 CABasicAnimation

KVC

keyPath实际上使用的是KVC的设计模式，keyPath本身的值是属性.后面的属性。

CABasicAnimation 属性

  • fromValue

fromValue is non-nil. Interpolates between fromValue and the current presentation value of the property.

指的是动画会从当前位置跳转到fromValue设值的位置再跳回来。

  • toValue

toValue is non-nil. Interpolates between the current value of keyPath in the target layer’s presentation layer and toValue.

指的是动画会直接跳转到指定的位置。

  • byValue

byValue is non-nil. Interpolates between the current value of keyPath in the target layer’s presentation layer and that value plus byValue.

指的是动画会从当前位置跳到当前位置叠加值后的位置。

  • fillMode

    fillMode源自CAMediaTiming，主要是与动画时间有关。

    fillMode的作用就是决定当前对象过了非active时间段的行为，比如动画的开始前，动画的结束后。如果是CAAnimation的动画，则需要将其removedOnCompletion设置为NO。要不然fillMode不起作用。

  • kCAFillModeForwards

    The receiver remains visible in its final state when the animation is completed.

    保留最终状态的变化，在动画完成后。

  • removedOnCompletion

    When YES, the animation is removed from the target layer’s animations once its active duration has passed. Defaults to YES.

    默认状态YES，指的应该是动作执行完成后是不是移除动作目标。
